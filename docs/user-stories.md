# User Stories – DriveAlb

## 1. Buyer (Car Buyer)

* As a buyer, I want to browse all available car listings so that I can explore different options.
* As a buyer, I want to search for cars by keywords so that I can quickly find specific vehicles.
* As a buyer, I want to filter cars by price, brand, model, and year so that I can narrow down my choices.
* As a buyer, I want to view detailed information about a car so that I can make an informed decision.
* As a buyer, I want to see images of the car so that I can better evaluate its condition.
* As a buyer, I want to contact the seller so that I can ask questions or arrange a purchase.

---

## 2. Seller (Car Owner)

* As a seller, I want to create an account so that I can manage my car listings.
* As a seller, I want to log in securely so that my account is protected.
* As a seller, I want to create a new car listing so that I can sell my vehicle.
* As a seller, I want to upload images of my car so that buyers can see it clearly.
* As a seller, I want to edit my listings so that I can update information if needed.
* As a seller, I want to delete my listings so that I can remove cars that are sold.

---

## 3. Admin (System Administrator)

* As an admin, I want to manage user accounts so that I can maintain control over the platform.
* As an admin, I want to remove inappropriate or fraudulent listings so that the platform remains trustworthy.
* As an admin, I want to monitor system activity so that I can ensure everything is functioning correctly.

---

## 4. System (Technical Perspective)

* As a system, I want to authenticate users securely so that user data is protected.
* As a system, I want to store car listings in a database so that data can be retrieved efficiently.
* As a system, I want to support fast search and filtering so that users have a smooth experience.

---

## 5. Future Enhancements (Optional)

* As a buyer, I want to save favorite cars so that I can review them later.
* As a buyer, I want to compare multiple cars so that I can choose the best option.
* As a seller, I want to see how many people viewed my listing so that I can understand interest.
* As an admin, I want to generate reports so that I can analyze platform usage.

---

## 6. Acceptance Criteria (Example)

### User Login

* User can enter email and password
* System validates credentials
* User is redirected to dashboard upon successful login

### Create Listing

* Seller fills in car details
* Seller uploads at least one image
* Listing is saved and visible to buyers
